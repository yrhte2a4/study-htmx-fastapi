# MCPサーバー使用ガイド

## 概要
このプロジェクトで有効化されているModel Context Protocol (MCP) サーバーの使用方法を説明します。

## 有効化されているMCPサーバー

### 1. Serena (コード解析・編集)
用途: プロジェクトのコード解析、シンボル検索、ファイル操作。コンテキストを節約するため積極的に活用してください

### 2. Context7 (ライブラリドキュメント)
用途: ライブラリやフレームワークの最新ドキュメント取得

### 3. Chrome DevTools (ブラウザ自動化)
用途: Webページの自動操作、テスト、デバッグ
注意: コンテキストを多く消費するため現在無効化しているため使用できません

### 4. Gemini CLI (AI検索・分析)
用途: 問題が発生した場合は積極的に利用し、アドバイスを求めてください
